# Plant Disease Detection Using CNN

## Project Overview
This project is a deep learning-based image classification system that detects plant diseases from leaf images. The model classifies potato plant leaves into three categories:
- Potato Early Blight
- Potato Late Blight
- Healthy Potato Leaf

The system uses Convolutional Neural Networks (CNN) for image classification.

## Technologies Used
- Python
- TensorFlow / Keras
- NumPy
- Matplotlib
- Google Colab
- Image Processing
- Convolutional Neural Networks (CNN)

## Dataset Classes
The model classifies images into the following categories:
- Potato___Early_blight
- Potato___Late_blight
- Potato___healthy

## Project Workflow
1. Collected plant leaf image dataset
2. Performed image preprocessing and resizing
3. Split dataset into training and testing sets
4. Built CNN model
5. Trained the model
6. Evaluated model accuracy and loss
7. Predicted disease from new leaf images

## Model Architecture
The CNN model includes:
- Convolutional layers
- ReLU activation
- Max pooling layers
- Flatten layer
- Dense layers
- Output layer for classification

## Features
- Plant disease detection from images
- CNN-based image classification
- Model training and evaluation
- Accuracy and loss visualization
- Prediction on new images
